Shift+화살표 = 5프레임씩 앞, 뒤로 이동
```


```
편집 과정
1. 모드 editing으로 변경
2. Ctrl+N > settings> Frame Size : Horizon / Vertical 해서 프레임 사이즈 조정
3. 프로젝트 패널 더블 클릭해서 영상, 오디오 소스 가져오기
4. 영상 마우스로 끌어와서 타임라인에 배치
5. 타임라인에서 컷 편집
6. 오디오 더블클릭
7. 오디오 적절하게 편집
8. 오디오 타임라인에 끌어오기
- 영상 개수에 따라 비디오, 오디오 v1/v2, a1, a2에 적절히 배치
9. export 눌러서 영상 추출
```
